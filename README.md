# Hermes Bot

AI-powered trading bot for Polymarket & Binance.

## Architecture

Railway (24/7) → Hermes Bot (Node.js) → AI API → Exchange

## Deploy

1. Push to GitHub
2. Import to Railway
3. Set env vars
4. Done

## Env Vars

Copy `.env.example` to `.env` and fill in your keys. On Railway, set them in the dashboard.
